# 垃圾邮件分类系统 — 实验报告 (v3)

> 数据集：24,000 封中文垃圾邮件（2019.10–2019.11）  
> 模型：BERT (chinese-roberta-wwm-ext)  
> 日期：2026-06-14  
> 更新：v3 — 重构分类体系，5 类均衡分布

---

## 1. 实验概述

### 1.1 问题定义

将 24,000 封已拦截的垃圾邮件分为 5 个子类别：

| 类别 | 英文标识 | 定义 |
|------|----------|------|
| 色情暴力 | adult_violence | 色情网站推广、成人内容、AV 视频、偷拍资源、裸聊约炮、暴力低俗内容 |
| 博彩营销 | commercial | 产品推广营销、展会搭建、招标采购、招聘广告、线上博彩、体育赌博、彩票投注 |
| 钓鱼邮件 | phishing | 账号安全验证、邮箱验证、密码窃取、虚假系统通知、文件/附件诱导下载、快递通知诈骗 |
| 发票财务 | finance | 代开发票、增值税发票、做账报销、财务询价、虚假财务报告、货款采购诈骗 |
| 学术推广 | academic | 学术会议征稿、论文润色发表、期刊投稿、培训课程、学术讲座、学位认证 |

### 1.2 版本演进

| 版本 | 类别体系 | macro-F1 | 最小类 | 极差 | Score | 关键变化 |
|------|----------|:------:|:------:|:---:|:----:|------|
| v1 | 色情/发票/营销/钓鱼/赌博 | 0.97 | 3.1% | 30.1 | 89.28 | 原始 5 类，赌博极少 |
| v2 | 同上 + from_name 特征 | 0.944 | 3.1% | 30.1 | 90.80 | 加入发件人特征，但分布未改善 |
| **v3** | **重构 5 类** | **0.916** | **13.3%** | **20.2** | — | 合并赌博入博彩营销，拆分色情，均衡分布 |

## 2. 数据

### 2.1 原始数据

24,000 封已拦截的垃圾邮件（2019.10–2019.11），来源为邮件网关日志。

### 2.2 分类体系设计

通过 LLM open 模式标注 500 条随机样本（DeepSeek-V3），得到 103 个细粒度标签。分析标签的构词规律后，采用优先级关键词分类器映射到 5 个均衡类别：

```
关键词优先级：
  发票/税务 → 发票财务
  赌博/博彩 → 博彩营销
  约炮/交友 → 色情暴力
  验证/钓鱼/密码 → 钓鱼邮件
  诈骗/欺诈 → 发票财务
  色情/成人 → 色情暴力
  学术/会议/论文 → 学术推广
  推广/营销/招标 → 博彩营销
```

### 2.3 训练数据

从 holdout 集（21,548 条）分层抽样 2,400 条，使用 DeepSeek-V3 以方案 1 类别定义进行封闭式标注。

| 指标 | 值 |
|------|-----|
| 抽样数量 | 2,400 |
| 标注成功率 | 100%（2,400/2,400） |
| LLM 费用 | ¥1.11 |
| 有效训练数据 | 2,388 条（置信度 ≥0.5） |

**训练数据分布：**

| 类别 | 数量 | 占比 |
|------|-----:|:----:|
| 色情暴力 | 822 | 34.4% |
| 发票财务 | 521 | 21.8% |
| 博彩营销 | 355 | 14.9% |
| 钓鱼邮件 | 350 | 14.7% |
| 学术推广 | 340 | 14.2% |

切分：train 1,670 / val 359 / test 359（分层抽样，种子 42）。

## 3. BERT 训练

### 3.1 配置

| 参数 | 值 |
|------|-----|
| 基座模型 | hfl/chinese-roberta-wwm-ext (110M) |
| 最大长度 | 256 tokens |
| 输入格式 | `{from_name} [SEP] {subject} [SEP] {content}` |
| Epochs | 5（Early Stopping patience=3） |
| Batch Size | 16 |
| Learning Rate | 2e-5 |
| Loss | Weighted CrossEntropy |
| 硬件 | NVIDIA RTX 4060 Laptop (8GB) |
| 训练时间 | 2.2 分钟 |

## 5. v3 改进总结

### 5.1 分类体系变化

| | v1/v2 | v3 |
|---|-------|----|
| 最小类 | 赌博 3.1% | 博彩营销 13.3% |
| 极差 | 30.1 | **20.2**（↓33%） |
| 接近零的类 | 1 个（赌博） | **0 个** |

### 5.2 对比

| 指标 | v1 | v2 | v3 |
|------|:--:|:--:|:--:|
| 类别体系 | 色情/发票/营销/钓鱼/赌博 | 同 v1 | 色情/博彩/钓鱼/发票/学术 |
| macro-F1 | 0.97 | 0.944 | 0.916 |
| 最小类 | 3.1% | 3.1% | **13.3%** |
| 极差 | 30.1 | 30.1 | **20.2** |
| LLM 验证 Score | 89.28 | 90.80 | **90.59** |
| LLM 验证 Accuracy | 94.8% | 95.6% | **95.8%** |
| 规则价值 | +3.5% | -5.1% | — |
| 推荐策略 | 融合 | 纯 BERT | **纯 BERT** |

macro-F1 的下降是预期的：v3 的类别边界发生了变化（博彩营销和学术推广从旧类别中拆分出来），测试集上的混淆主要在 commercial ↔ phishing 之间，这反映了新类别体系中"博彩营销"和"钓鱼诱导"之间的真实模糊性。
